10.1 MAC关闭指定端口

MAC 关闭指定端口

先执行如下命令：

lsof -i:端口号

会有类似下面的结果：

COMMAND     PID       USER   FD   TYPE             DEVICE SIZE/OFF NODE NAME
Google    22737 bookkilled   78u  IPv6 0x1cb34cea0d8ed947      0t0  TCP localhost:54783->localhost:hbci (ESTABLISHED)
node      96717 bookkilled   13u  IPv6 0x1cb34cea04e85f07      0t0  TCP *:hbci (LISTEN)
node      96717 bookkilled   14u  IPv6 0x1cb34cea0d8ee407      0t0  TCP localhost:hbci->localhost:54783 (ESTABLISHED)

然后执行杀端口：

kill -9 96717

结束进程就搞定了···
